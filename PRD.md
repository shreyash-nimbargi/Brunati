# BRUNATI | Luxury Fragrance

## Project Overview
A premium e-commerce platform for luxury fragrances, featuring a sleek React frontend and a robust Express backend.

## Tech Stack
- **Frontend**: React (Vite), React Router, Tailwind CSS (optional), Vanilla CSS.
- **Backend**: Node.js, Express, MongoDB (Mongoose).
- **Authentication**: Custom JWT-based auth flow.

## Key Features
- Dynamic product catalog.
- Interactive shopping experience.
- Admin dashboard for product management.
